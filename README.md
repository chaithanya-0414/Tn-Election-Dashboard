# TN Election 2026 Interactive Dashboard

An interactive, premium "Dark Mode" data visualization dashboard tracking the multi-cornered contest of the upcoming 2026 Tamil Nadu Legislative Assembly Election.

## Overview
This interactive dashboard analyzes aggregate opinion polling data to project the highly anticipated 2026 Tamil Nadu State Election. It visualizes the tightening race between the major political alliances and specifically tracks the "spoiler" impact of the emerging Tamilaga Vettri Kazhagam (TVK).

**View the Live Dashboard Here:** [https://chaithanya-0414.github.io/Tn-Election-Dashboard/](https://chaithanya-0414.github.io/Tn-Election-Dashboard/)

## Key Features
*   **Aesthetic & Modern:** Employs a premium dark theme, glassmorphic UI elements (backdrop blurs), and dynamic gradients for high-impact visual delivery.
*   **Seat Range Projections:** Utilizes a custom Chart.js configuration to visualize the projected seat *ranges* for each alliance, including a dynamic "Majority Mark" reference line.
*   **Regional Filtering:** Interactive dropdown allows users to drill down from statewide aggregate data into specific battleground regions (Northern TN, Western/Kongu Belt, Delta, and Southern TN).
*   **Vote Share Analysis:** Includes a detailed breakdown of the projected popular vote to highlight the multi-cornered electoral mechanics.

## Tech Stack
*   **HTML5**
*   **Tailwind CSS:** Used via CDN for rapid, responsive utility-class styling and advanced UI design (Glassmorphism, gradients).
*   **Chart.js:** Custom-configured to match the dark aesthetic and render complex datasets (ranges and standard datasets).
*   **Google Fonts:** Utilizing `Outfit` for display integers/metrics and `Inter` for general readability.

## Disclaimer
This project visualizes **estimated projections** based on aggregate pre-poll surveys and opinion data. These are **not actual election results**. This dashboard was designed strictly for strategic analysis and data visualization purposes.

## Local Setup
To run this project locally, simply clone the repository and open `index.html` in any modern web browser. No complex build tools or local servers are required.
